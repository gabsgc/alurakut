## :computer: Technologies

This project was made using the following technologies:
- React.JS
- Next.JS
- JavaScript

## ⛏ Dependencies
- Yarn
- DatoCMS 
- Nookies
- JWT

## :construction_worker: How to run

```
# Clone Repository
$ git clone https://github.com/gabsgc/alurakut

# Go to folder
$ cd alurakut

# Install Dependencies
$ yarn

# Run Web Server
$ yarn dev

```
